## Features

- 설문지 생성
- 질문 생성 (택스트, 객관식, 체크박스)
- 설문지 공유
- 응답 결과 확인 (요약, 질문, 개별보기)
- 응답 결과 .csv형식 다운로드

## 개발할 기능

- 설문지 검색

## Tech and libraries

- django 3.2.6
- jquery
- sqlite 3.36.0
- toastr
- jquery.cookie
- chart.js

## Getting started

- use git clone.

```bash
$ git clone https://github.com/morethanmin/Fooorm.git
$ cd Fooorm
$ python3 manage.py runserver (or python manage.py runserver)
```

- open http://localhost:8000/
